# BondPricer: Quant Tools for Fixed Income Investing

BondPricer is a Python project providing quantitative tools for fixed income investing. The current focus is on pricing US Treasury bonds using robust, transparent, and extensible code.

## Features
- Pricing of US Treasury bonds (clean and dirty price, accrued interest)
- Yield curve bootstrapping and interpolation
- Cash flow generation and schedule calculation
- Example Jupyter notebooks for hands-on learning

## Upcoming Workflow
- Risk and return analysis for fixed income portfolios
- Scenario analysis and sensitivity (duration, convexity)
- Tools for holding period return and reinvestment risk

This project is designed for both learning and practical application in quantitative finance.
